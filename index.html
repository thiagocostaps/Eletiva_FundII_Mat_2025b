Sobre a Eletiva

A eletiva de Programação tem como objetivo introduzir os alunos ao desenvolvimento web, estimulando criatividade, autonomia e raciocínio lógico através de pequenos projetos práticos. Durante a eletiva os alunos do Fundamental II iniciaram seus estudos pela lógica de programação, entendendo algoritmos, sequência, seleção e repetição. Em seguida, exploraram o editor VS Code para organização e construção de códigos, aprendendo a criar páginas web com HTML e estilização com CSS, além de aplicar efeitos interativos utilizando JavaScript. Também criaram contas no GitHub, publicaram seus projetos e aprenderam a usar o GitHub Pages para hospedar seus sites. No universo do Scratch, os estudantes aprenderam desde os comandos básicos até a criação de projetos interativos completos. Trabalharam com movimentação, cenários, sons, variáveis, operadores, sensores, paralelismo e blocos de controle. Desenvolveram projetos individuais e em dupla, praticando tomada de decisão, uso de loops, interação com mouse e teclado, criação de animações e aplicação de efeitos. Além disso, aprenderam a identificar e corrigir erros nos próprios códigos.
